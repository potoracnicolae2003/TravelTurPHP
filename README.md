# ExploreX Travel - Agenție de Turism

Site web modern pentru o agenție de turism, cu logare reală și oferte turistice.

## Funcționalități
- Pagina principală de prezentare
- Logare cu utilizator și parolă (autentificare reală)
- Oferte vizibile doar după logare
- Bază de date SQLite pentru utilizatori

## Instalare și rulare

1. **Deschide terminalul în folderul `TravelTur`**
2. Instalează dependențele Node.js:
   ```
npm install
   ```
3. Pornește serverul:
   ```
npm start
   ```
4. Accesează site-ul la adresa:
   ```
http://localhost:3000
   ```

## Date de test (utilizator implicit):
- Utilizator: `admin`
- Parolă: `parola123`

## Structură fișiere importante
- `Untitled-1.html` - Pagina principală
- `login.html` - Pagina de logare
- `oferte.html` - Pagina cu oferte (protejată)
- `style.css` - Stilizare modernă
- `script.js` - Validare și interactivitate
- `server.js` - Backend Node.js (Express)
- `users.db` - Baza de date SQLite (se generează automat)
